# Netflix Movies & TV Shows Analysis  
_Анализ контентных предпочтений для платформы Netflix_

## Цель проекта  
Исследовать динамику и предпочтения контента на платформе Netflix:  
- по годам выпуска  
- по странам производства  
- по жанрам.  
Выполнены: EDA, визуализации (seaborn + plotly), кластеризация по жанрам и регионам, сравнение активности по странам.

## Данные  
- Файл: `netflix_titles.csv`  
- Колонки: `show_id`, `type` (Movie / TV Show), `title`, `director`, `cast`, `country`, `date_added`, `release_year`, `rating`, `duration`, `listed_in` (жанры), `description`  
- Фильтрация: релизы с **1990 года и позже**  
- Предобработка:  
  - Заполнение пропусков (`country`, `listed_in`)  
  - Разделение жанров (`listed_in`) и стран (`country`)  
  - Создание признаков: `year_added`, `genre_list`, `country_list`

## Этапы и методы  
1. **EDA (Exploratory Data Analysis)**:  
   - Кол-во релизов по годам  
   - Топ-стран по объёму контента  
   - Топ-жанров  
   - Динамика Movies vs TV Shows  
   - Heatmap активности стран × годов  
   - Treemap жанров × типов (Movie/TV Show)  
2. **Визуализации**:  
   - Статические (matplotlib, seaborn)  
   - Интерактивные (plotly)  
3. **Кластеризация**:  
   - One-hot кодирование жанров и стран  
   - PCA для снижения размерности  
   - KMeans (6 кластеров)  
   - Визуализация кластеров (PCA2D)  
4. **Выводы и интерпретации**:  
   - Основные страны-производители контента  
   - Популярность жанров  
   - Видимые тренды по годам  
   - Распределение кластеров контента

## Как запустить  
1. Открой `analysis_notebook.ipynb` в Google Colab или локально.  
2. Убедись, что файл `netflix_titles.csv` находится в рабочей папке.  
3. Запусти все ячейки:  
   - Предобработка данных  
   - EDA и визуализации  
   - Кластеризация  
4. Графики будут отображены прямо и/или сохранены в папке `figures/`.

- **Heatmap активности контента по ТОП-20 странам × годам**

## Основные выводы  
- США, Великобритания и Индия — ведущие страны-поставщики контента на Netflix.  
- Пик активности добавлений контента пришёлся на конец 2010-х годов.  
- Жанры “Драма”, “Комедия”, “Триллер” — наиболее распространены.  
- Кластеризация выявила 6 различных групп контента по жанрам и регионам: например, «англоязычные драмы», «индийский массовый рынок», «азиатские сериалы» и др.  
- Большинство стран за пределами топ-20 имеют очень низкую активность (1–5 добавлений в год).
